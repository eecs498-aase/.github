# Applied Agentic Software Engineering

> Build real multi-agent systems. Not toy demos.

**EECS 498-016** is a 4-credit course at the University of Michigan that teaches students to build production-grade agent orchestration systems from scratch — inspired by open-source platforms like [OpenClaw](https://openclaw.ai).

## What You'll Learn

Students progress through three phases of increasing complexity:

| Phase | Focus | What You Build |
|-------|-------|----------------|
| **01** — AI-Assisted Development | Principled AI coding with Context, Model, Prompt | AI coding portfolio using Aider |
| **02** — Scripted Agent Workflows | Multi-agent protocols, state management, LLM integration | Working multi-agent orchestration system |
| **03** — Build an Orchestrator | Task systems, tool registries, delegation, evaluation | Production-grade agent platform (from scratch) |

**No exams.** All projects, lab checkoffs, and an oral defense.

## Course Details

- **Term:** Fall 2026
- **Credits:** 4
- **Prerequisites:** EECS 281 + EECS 201 (or equivalent)
- **Format:** Tue/Thu lectures + weekly lab sections
- **Instructors:** Marcus Darden, Rafe Symonds (GSI)
